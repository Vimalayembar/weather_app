# 🌤️ Weather App

A modern React Weather Application built using React, Vite, JavaScript, CSS, and OpenWeather API.

## 🚀 Features

- 🌡️ Current weather data
- 📅 5-day weather forecast
- ⏰ Hourly forecast
- 📍 Current location weather
- ⭐ Saved locations
- 🕘 Recent searches
- 🌤️ Dynamic weather icons
- ⌨️ Enter key search
- ❌ Error handling
- ⏳ Loading state
- 💾 Local storage persistence

## 🛠️ Tech Stack

- React
- JavaScript
- Vite
- CSS
- OpenWeather API

## ▶️ Run Locally

Clone the project:

```bash
git clone https://github.com/Vimalayembar/weather_app.git
````

Go to project folder:

```bash
cd weather_app
```

Install dependencies:

```bash
npm install
```

Run app:

```bash
npm run dev
```
